About This Effect

This interactive background simulates an electric static field using HTML5 Canvas and JavaScript. The particles respond to your cursor movement and clicks, creating dynamic energy patterns.

How to Interact

Move your cursor to influence the electric field. Click to create energy bursts. Use the control buttons to adjust intensity, change colors, or reset the effect to its default state.

Technical Details

The effect uses particle physics with repulsion/attraction forces. Each particle has electrical properties that create the static appearance. Performance is optimized using requestAnimationFrame.
